Semantica (Esta dentro de header)

La etiqueta <header> define una cabecera para un documento o sección.
La etiqueta <nav> define un conjunto de enlaces de navegación.
La etiqueta <main> especifica el contenido principal o importante del documento.
La etiqueta <section> sirve para agrupar contenido relacionado.
La etiqueta <article> define un artículo independiente.
La etiqueta <aside> define contenido que debe mostrarse aparte del contenido de la página web.
La etiqueta <summary> define un encabezado visible para el elemento <details>. La etiqueta <details> define información adicional que el usuario puede mostrar u ocultar.

La etiqueta <time> define una fecha/hora. La fecha de publicación de este documento es 22 de octubre del 2023.

La etiqueta <footer> define un pie de página para un documento o sección.
