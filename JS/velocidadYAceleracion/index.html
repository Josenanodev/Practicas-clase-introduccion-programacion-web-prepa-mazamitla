Velocidad y Aceleracion

Velocidad

La velocidad es una magnitud física de carácter vectorial que expresa el desplazamiento de un objeto por unidad de tiempo. Se representa con la letra v.

Podemos calcular la velocidad con respecto a la posicion si sabemos las siguientes variables:

  • Posicion inicial = x 0
  • Posicion final = x 1
  • Tiempo inicial = t 0
  • Tiempo final = t 1

La formula para calcular la velocidad es la siguiente:

v = distancia tiempo = x 1 - x 0 t 1 - t 0

Aceleracion

La aceleración es una magnitud física de carácter vectorial que expresa la variación de la velocidad de un objeto por unidad de tiempo. Se representa con la letra a.

Podemos calcular la aceleracion con respecto a la velocidad si sabemos las siguientes variables:

  • Velocidad inicial = v i
  • Velocidad final = v f
  • Tiempo inicial = t i
  • Tiempo final = t f

La formula para calcular la aceleracion es la siguiente:

a = velocidad tiempo = v f - v i t f - t i
